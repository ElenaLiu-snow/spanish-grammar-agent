# Género del Sustantivo
> NGLE Reference: Chapter 2
> Source: Real Academia Española — NGLE

## 基本规则

西语名词分阳性和阴性两大类。

### 阳性名词常见词尾
| 词尾 | 例词 | 例外 |
| --- | --- | --- |
| -o | `el libro`, `el perro` | `la mano`, `la radio` |
| -or | `el amor`, `el color` | `la flor`, `la labor` |
| -aje | `el viaje`, `el paisaje` | — |
| -ema/-ama | `el tema`, `el programa` | — |
| -án/-ón | `el capitán`, `el corazón` | — |

### 阴性名词常见词尾
| 词尾 | 例词 | 例外 |
| --- | --- | --- |
| -a | `la casa`, `la mesa` | `el día`, `el mapa` |
| -ción/-sión | `la canción`, `la decisión` | — |
| -dad/-tad | `la ciudad`, `la libertad` | — |
| -tud | `la actitud`, `la juventud` | — |
| -ez/-eza | `la vejez`, `la belleza` | — |
| -umbre | `la costumbre`, `la cumbre` | — |

## 特殊规则

### 以 a-/ha- 开头的阴性名词
为避免元音重复，单数用阳性冠词 `el`，但词本身仍是阴性：

| 正确 | 错误 |
| --- | --- |
| `el agua fría` | ~~la agua fría~~ |
| `el hacha afilada` | ~~la hacha afilada~~ |
| `el aula vacía` | ~~la aula vacía~~ |
| `mucha agua` (冠词不受限) | — |

### 共性名词 (sustantivos comunes)
同一形式可指男女性别，通过冠词区分：
- `el/la estudiante`, `el/la pianista`, `el/la periodista`

### 模糊性名词 (sustantivos ambiguos)
可带阳或阴性冠词且意义相同：
- `el mar / la mar`, `el azúcar / la azúcar`

### 异义名词（换性别换意思）
| 阳性 | 阴性 |
| --- | --- |
| `el capital`（资本） | `la capital`（首都） |
| `el corte`（切口） | `la corte`（宫廷） |
| `el frente`（前线） | `la frente`（额头） |
| `el orden`（顺序） | `la orden`（命令） |
| `el cura`（神父） | `la cura`（治愈） |
