# 陈述式时态体系 (Modo Indicativo)
> NGLE Reference: Chapters 23-24
> Source: Real Academia Española — NGLE

## 陈述式时态总览

| 时态 | 西语名称 | 用法 |
| --- | --- | --- |
| 现在时 | Presente | 当前习惯/事实/进行 |
| 现在完成时 | Pret. Perfecto Compuesto | 与现在相关的过去 |
| 简单过去时 | Pret. Indefinido | 完成的过去动作 |
| 过去未完成时 | Pret. Imperfecto | 过去背景/习惯 |
| 过去完成时 | Pret. Pluscuamperfecto | 过去的过去 |
| 先过去时 | Pret. Anterior | 紧接着的过去（罕用） |
| 简单将来时 | Futuro Simple | 将来动作/推测 |
| 将来完成时 | Futuro Compuesto | 将来某时前完成 |
| 简单条件式 | Condicional Simple | 礼貌/假设/过去将来 |
| 复合条件式 | Condicional Compuesto | 未实现的过去假设 |

## Presente（现在时）的特殊用法

1. **现在习惯**：`**Leo** antes de dormir.`
2. **现在进行**：`¿Qué **haces**?`
3. **永恒事实**：`La Tierra **gira** alrededor del Sol.`
4. **历史现在**：`Colón **llega** a América en 1492.`
5. **将来计划**：`Mañana **salgo** para Madrid.`
6. **条件/假设**：`Si **llueve**, no salgo.`

## Futuro Simple 的推测用法

| 用法 | 例句 |
| --- | --- |
| 现在推测 | `¿Dónde **estará** Juan? — **Estará** en casa.` |
| 近似数量 | `**Tendrá** unos 30 años.` |
| 让步 | `**Será** muy inteligente, pero no estudia.` |

## Condicional Simple 的用法

1. **礼貌请求**：`¿**Podrías** ayudarme?`
2. **过去将来**：`Dijo que **vendría**.`
3. **假设条件**：`Si tuviera dinero, **viajaría**.`
4. **过去推测**：`Serían las tres cuando llegó.`
