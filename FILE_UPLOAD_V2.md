# 文件上传功能更新说明 v2

## 改进内容

### 1. 用户界面优化

**之前**: 用户消息中显示完整的文档内容，导致对话框很长很冗长

**现在**: 用户消息中只显示文件名，例如：
```
[文件: Unidad 16 conjugacion de verbos.docx]
批改这份练习并讲解错误
```

**后台**: 完整的文档内容仍然会发送给 AI 进行分析

### 2. Word 表格识别改进

**之前**: 表格内容被简单处理为行文本，格式混乱

**现在**: 表格转换为 Markdown 格式，保留结构

**示例**:

原始 Word 表格:
```
| 动词   | 陈述式现在时        | 过去未完成时        |
|--------|-------------------|-------------------|
| Hablar | yo hablo           | yo hablaba        |
| Comer  | yo como            | yo comía          |
```

### 支持的文件格式

| 文件类型 | 扩展名 | 功能 |
|---------|--------|------|
| Word 文档 | .docx | ✅ 提取文本和表格（Markdown 格式）|
| Markdown | .md | ✅ 保留 Markdown 结构 |
| 纯文本 | .txt | ✅ 读取内容 |
| 图片 | .png, .jpg, .jpeg, .gif, .webp | ⚠️ 开发中 |

### 使用方法

1. **上传文件** - 点击"上传文件"按钮
2. **查看文件信息** - 显示文件名和字符数
3. **输入问题**（可选）- 例如："分析这个文档中的语法错误"
4. **发送** - AI 会分析完整的文档内容

### 技术实现

- 前端: 分离显示内容和实际发送内容
- 后端: 使用 python-docx 遍历文档元素，保持段落和表格的顺序
- 表格转换: Word 表格 → Markdown 表格格式
- token 限制: max_tokens = 8192

### 示例对话

**用户输入**:
```
[文件: Unidad 16 conjugacion de verbos.docx]
批改这份练习并讲解错误
```

**AI 分析**: AI 会收到完整的文档内容，包括：
- 学生信息
- 练习要求
- 动词变位表格
- 并提供详细的批改和讲解
