# 条件句 (Oraciones Condicionales)
> NGLE Reference: Chapter 47
> Source: Real Academia Española — NGLE

## 条件句三大类型

### 第一类：真实条件 (Si + Presente de Indicativo)
表达可能的、真实的条件。

```
Si + presente de indicativo → presente / futuro / imperativo
```

| 例句 | 翻译 |
| --- | --- |
| `Si **llueve**, no **salgo**.` | 如果下雨，我就不出去。 |
| `Si **estudias**, **aprobarás**.` | 如果你学习，就会通过。 |
| `Si **quieres**, **ven** conmigo.` | 如果你想，就跟我来。 |

### 第二类：假设条件 (Si + Pret. Imperfecto de Subjuntivo)
表达不太可能或假设的条件。

```
Si + pret. imperfecto de subj. → condicional simple
```

| 例句 | 翻译 |
| --- | --- |
| `Si **tuviera** dinero, **viajaría** por el mundo.` | 如果我有钱，就环游世界。 |
| `Si **fuera** tú, no lo **haría**.` | 如果我是你，我不会这样做。 |
| `Si **estudiaras** más, **sacarías** mejores notas.` | 如果你多学习，会取得更好的成绩。 |

### 第三类：过去未实现条件 (Si + Pret. Pluscuamperfecto de Subj.)
表达过去没有实现的条件（不可改变）。

```
Si + pret. pluscuamperfecto de subj. → condicional compuesto
                               → pret. pluscuamperfecto de subj.
```

| 例句 | 翻译 |
| --- | --- |
| `Si **hubieras estudiado**, **habrías aprobado**.` | 如果你当时学习了，就通过了。 |
| `Si **hubiera sabido**, **habría venido** antes.` | 如果我当时知道，就早点来了。 |
| `Si no **hubiera llovido**, **habríamos salido**.` | 如果当时没下雨，我们就出去了。 |

## 其他条件连接词

| 连接词 | 例句 |
| --- | --- |
| `como + subj.` | `Como no **vengas**, me voy.` |
| `a condición de que + subj.` | `Te ayudo a condición de que **estudies**.` |
| `siempre que + subj.` | `Puedes salir siempre que **termines**.` |
| `en caso de que + subj.` | `Llámame en caso de que **necesites** ayuda.` |
| `a no ser que + subj.` | `No voy a no ser que me **invites**.` |
| `con tal de que + subj.` | `Haré lo que sea con tal de que **vengas**.` |
