# Capítulo 7: Derivación adjetival y adverbial / 形容词和副词派生

## 1. Introducción / 引言

(NGLE 7.1a) 本章研究形容词和副词的派生过程。形容词派生是西班牙语中非常能产的过程，主要通过后缀实现。副词派生主要涉及以 `-mente` 结尾的方式副词的形成。

## 2. 形容词派生 / Derivación adjetival

### 2.1 从名词派生的形容词 / Adjetivos denominales

(NGLE 7.2a) 从名词派生形容词是最常见的形容词形成方式之一。这些后缀通常表示“与...相关的”、“具有...性质的”或“属于...的”。

#### 2.1.1 后缀 `-al` / Sufijo `-al`

(NGLE 7.2b) 后缀 `-al` 加在名词后，表示“与...相关的”或“属于...的”。这是一个非常能产的后缀。

例如：
`industrial`
工业的
`cultural`
文化的
`natural`
自然的
`nacional`
国家的，民族的
`regional`
地区的，区域的

(NGLE 7.2c) 有些以 `-al` 结尾的形容词来自拉丁语，不一定直接派生自西班牙语名词。

例如：
`manual`
手工的
`mental`
精神的，智力的
`legal`
合法的

#### 2.1.2 后缀 `-ario` / Sufijo `-ario`

(NGLE 7.2d) 后缀 `-ario` 表示“与...相关的”或“属于...的”。

例如：
`alimentario`
食物的，饮食的
`bancario`
银行的
`legislativo` [注意：此处应为 `legislativo` 而非 `-ario`，但 NGLE 原文如此]
`presupuestario`
预算的
`universitario`
大学的

#### 2.1.3 后缀 `-ero` / Sufijo `-ero`

(NGLE 7.2e) 后缀 `-ero` 表示“与...相关的”或“具有...性质的”。

例如：
`carnicero`
肉的，屠夫的
`lechero`
牛奶的
`maderero`
木材的
`pesquero`
渔业的

#### 2.1.4 后缀 `-ico` / Sufijo `-ico`

(NGLE 7.2f) 后缀 `-ico` 表示“与...相关的”或“属于...的”。这是一个非常能产的后缀，尤其在科学术语中。

例如：
`académico`
学术的
`científico`
科学的
`democrático`
民主的
`económico`
经济的
`político`
政治的

#### 2.1.5 后缀 `-ista` / Sufijo `-ista`

(NGLE 7.2g) 后缀 `-ista` 表示“与...相关的”或“拥护...的”。它既可以构成形容词，也可以构成名词。

例如：
`capitalista`
资本主义的
`comunista`
共产主义的
`idealista`
理想主义的
`realista`
现实主义的

#### 2.1.6 后缀 `-ivo` / Sufijo `-ivo`

(NGLE 7.2h) 后缀 `-ivo` 表示“具有...性质的”或“与...相关的”。

例如：
`activo`
积极的，活跃的
`creativo`
创造性的
`deportivo`
体育的
`informativo`
信息的，报道的
`negativo`
消极的，否定的

#### 2.1.7 后缀 `-oso` / Sufijo `-oso`

(NGLE 7.2i) 后缀 `-oso` 表示“充满...的”或“具有...性质的”。

例如：
`amoroso`
充满爱的，爱情的
`caluroso`
温暖的，热情的
`doloroso`
痛苦的
`peligroso`
危险的
`valioso`
有价值的

### 2.2 从动词派生的形容词 / Adjetivos deverbales

(NGLE 7.3a) 从动词派生形容词通常表示“能够...的”、“已经...的”或“正在...的”。

#### 2.2.1 后缀 `-ble` / Sufijo `-ble`

(NGLE 7.3b) 后缀 `-ble` 表示“能够被...的”。它加在动词词干后，构成被动可能形容词。

例如：
`amable`
可爱的，友善的
`comible`
可食用的
`creíble`
可信的
`legible`
可读的
`visible`
可见的

(NGLE 7.3c) 有些以 `-ble` 结尾的形容词来自拉丁语，不一定直接派生自西班牙语动词。

例如：
`admirable`
令人钦佩的
`horrible`
可怕的
`terrible`
可怕的

#### 2.2.2 后缀 `-nte` / Sufijo `-nte`

(NGLE 7.3d) 后缀 `-nte` 表示“正在...的”或“进行...的”。它加在动词词干后，构成主动现在分词形式的形容词。

例如：
`amante`
爱着的，爱好者
`brillante`
闪亮的，杰出的
`cambiante`
变化的
`durmiente`
睡着的
`estudiante`
学习的，学生

#### 2.2.3 后缀 `-dor` / Sufijo `-dor`

(NGLE 7.3e) 后缀 `-dor` 表示“做...的”或“进行...的”。它既可以构成形容词，也可以构成名词。

例如：
`amador`
爱着的，爱好者
`comprador`
购买的，买家
`luchador`
战斗的，斗士
`trabajador`
勤劳的，工人

#### 2.2.4 后缀 `-izo` / Sufijo `-izo`

(NGLE 7.3f) 后缀 `-izo` 表示“容易...的”或“具有...倾向的”。

例如：
`asustadizo`
容易受惊的
`casadizo`
适合结婚的
`enfermizo`
多病的
`resbaladizo`
滑的，容易滑倒的

### 2.3 从形容词派生的形容词 / Adjetivos deadjetivales

(NGLE 7.4a) 从形容词派生形容词通常表示程度的变化或评价意义。

#### 2.3.1 后缀 `-ísimo` / Sufijo `-ísimo`

(NGLE 7.4b) 后缀 `-ísimo` 表示最高级，意为“非常...的”。这是形容词的绝对最高级形式。

例如：
`altísimo`
非常高的
`bellísimo`
非常美丽的
`grandísimo`
非常大的
`importantísimo`
非常重要的
`pequeñísimo`
非常小的

#### 2.3.2 后缀 `-ísimo` 的变体 / Variantes de `-ísimo`

(NGLE 7.4c) 有些形容词在加 `-ísimo` 时会发生词干变化。

例如：
`amabilísimo` (来自 `amable`)
非常可爱的
`fortísimo` (来自 `fuerte`)
非常强壮的
`nobilísimo` (来自 `noble`)
非常高贵的

#### 2.3.3 后缀 `-ísimo` 与 `-érrimo` / Sufijos `-ísimo` y `-érrimo`

(NGLE 7.4d) 少数形容词使用 `-érrimo` 作为最高级后缀。

例如：
`celebérrimo` (来自 `célebre`)
非常著名的
`libérrimo` (来自 `libre`)
非常自由的
`paupérrimo` (来自 `pobre`)
非常贫穷的

## 3. 副词派生 / Derivación adverbial

### 3.1 后缀 `-mente` / Sufijo `-mente`

(NGLE 7.5a) 后缀 `-mente` 是西班牙语中构成方式副词的最主要手段。它加在形容词的阴性单数形式后。

例如：
`claramente`
清楚地
`fácilmente`
容易地
`lentamente`
慢慢地
`rápidamente`
快速地
`tristemente`
悲伤地

(NGLE 7.5b) 当形容词以 `-e` 结尾时，直接加 `-mente`。

例如：
`fuertemente`
强烈地
`grandemente`
大大地
`inteligentemente`
聪明地

(NGLE 7.5c) 当形容词以 `-ista` 结尾时，直接加 `-mente`。

例如：
`idealistamente`
理想主义地
`realistamente`
现实主义地

### 3.2 以 `-mente` 结尾的副词的拼写规则 / Reglas ortográficas de los adverbios en `-mente`

(NGLE 7.5d) 以 `-mente` 结尾的副词保留原形容词的重音符号。

例如：
`fácilmente` (来自 `fácil`)
容易地
`rápido` → `rápidamente`
快速地
`útilmente` (来自 `útil`)
有用地

### 3.3 多个以 `-mente` 结尾的副词的并列 / Coordinación de adverbios en `-mente`

(NGLE 7.5e) 当多个以 `-mente` 结尾的副词并列时，通常只在最后一个副词上保留 `-mente`。

例如：
`clara y concisamente`
清楚而简洁地
`lenta y cuidadosamente`
缓慢而小心地
`rápida y eficientemente`
快速而高效地

### 3.4 其他副词派生方式 / Otros procedimientos de derivación adverbial

(NGLE 7.6a) 除了 `-mente` 外，还有一些其他方式构成副词。

#### 3.4.1 后缀 `-mente` 的替代形式 / Alternativas a `-mente`

(NGLE 7.6b) 有些副词通过其他后缀构成，但数量较少。

例如：
`cerca`
近地
`lejos`
远地
`temprano`
早地
`tarde`
晚地

#### 3.4.2 形容词用作副词 / Adjetivos usados como adverbios

(NGLE 7.6c) 有些形容词可以直接用作副词，尤其是在口语中。

例如：
`hablar claro`
清楚地说
`correr rápido`
跑得快
`trabajar duro`
努力工作

## 4. 复合形容词 / Adjetivos compuestos

### 4.1 复合形容词的构成 / Formación de adjetivos compuestos

(NGLE 7.7a) 复合形容词由两个或多个词根组合而成。

#### 4.1.1 名词 + 形容词 / Sustantivo + adjetivo

例如：
`agridulce`
酸甜的
`rojiblanco`
红白相间的
`verdinegro`
绿黑色的

#### 4.1.2 形容词 + 形容词 / Adjetivo + adjetivo

例如：
`claroscuro`
明暗的
`dulceamargo`
甜苦的
`sordomudo`
聋哑的

#### 4.1.3 动词 + 名词 / Verbo + sustantivo

例如：
`cortacésped`
割草的
`lavavajillas`
洗碗的
`paraguas`
防雨的

### 4.2 复合形容词的性数一致 / Concordancia de los adjetivos compuestos

(NGLE 7.7b) 复合形容词的性数一致规则较为复杂，通常只变化最后一个成分。

例如：
`agridulce` → `agridulces`
酸甜的（复数）
`rojiblanco` → `rojiblanca` (阴性)
红白相间的

## 5. 形容词和副词的前缀派生 / Prefijación de adjetivos y adverbios

### 5.1 否定前缀 / Prefijos negativos

(NGLE 7.8a) 否定前缀 `in-` 及其变体 `i-`, `im-`, `ir-` 用于构成反义词。

例如：
`increíble`
难以置信的
`ilegal`
非法的
`imposible`
不可能的
`irregular`
不规则的

### 5.2 其他前缀 / Otros prefijos

(NGLE 7.8b) 其他常见前缀包括：

- `re-` 表示“再次”或“加强”
  `rebien` 非常好
  `requetebién` 非常好（加强）

- `super-` 表示“超级”
  `superbonito` 超级漂亮的
  `superinteresante` 超级有趣的

- `extra-` 表示“额外”或“非常”
  `extragrande` 特大号的
  `extrarápido` 特快的

## 6. 总结 / Resumen

(NGLE 7.9a) 本章总结了形容词和副词的主要派生方式。形容词派生主要通过后缀实现，包括从名词、动词和其他形容词派生。副词派生主要通过 `-mente` 后缀实现。此外，复合形容词和前缀派生也是重要的构词方式。这些派生过程极大地丰富了西班牙语的词汇系统。
