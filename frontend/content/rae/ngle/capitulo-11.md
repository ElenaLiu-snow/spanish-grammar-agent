# Capítulo 11: Composición / 复合构词

## 1. Concepto de composición / 复合构词的概念

`composición`是一种`procedimiento morfológico`，它由两个或多个`bases léxicas`结合构成一个新词。(NGLE 11.1a)

复合构词是一种形态学过程，它将两个或多个词基结合以形成新词。

例如：
`hispano + hablante > hispanohablante`
西班牙语 + 说话者 > 说西班牙语的人

`hierba + buena > hierbabuena`
草 + 好 > 薄荷

### 1.1 Características generales / 一般特征

复合词的特点是具有整体意义，并且在许多情况下，其语法行为有别于相应的句法序列。（`NGLE` 11.1b）

复合词的特点是具有整体意义，并且在许多情况下，其语法行为与等效的句法序列不同。

例如：
`hispanoamericano` (compuesto) vs. `americano hispano` (secuencia sintáctica)
西班牙裔美洲人（复合词） vs. 西班牙裔的美洲人（句法序列）

## 2. Clases de compuestos / 复合词的类别

### 2.1 Compuestos propios e impropios / 固有复合词与非固有复合词

区分两种基本复合词类型：`固有复合词`和`非固有复合词`。（NGLE 11.2a）

区分两种基本类型的复合词：固有复合词和非固有复合词。

#### 2.1.1 Compuestos propios / 固有复合词

`固有复合词`是指通过两个或多个词基的图形结合形成的，且没有任何连接元素介入的复合词。（NGLE 11.2b）

固有复合词是指两个或多个词基在书写上直接结合，中间没有任何连接元素。

例如：
`hispano + árabe > hispanoárabe`
西班牙 + 阿拉伯 > 西班牙-阿拉伯的

`agua + fuerte > aguafuerte`
水 + 强 > 蚀刻版画

#### 2.1.2 Compuestos impropios / 非固有复合词

不纯复合词是指由两个或以上保持图形独立性但作为一个词汇单位发挥作用的词基结合而成的复合词。(NGLE 11.2c)

非固有复合词是指两个或多个词基在书写上保持独立，但作为一个词汇单位发挥作用。

例如：
`coche cama`
卧铺车

`hombre rana`
蛙人

### 2.2 Compuestos cultos y populares / 书面复合词与通俗复合词

#### 2.2.1 Compuestos cultos / 书面复合词

`学术复合词`由源自拉丁语或希腊语的元素构成，这些元素在西班牙语中不作为独立单词存在。（NGLE 11.3a）

书面复合词由源自拉丁语或希腊语的元素构成，这些元素在西班牙语中不作为独立单词存在。

例如：
`antropo-` (hombre) + `-fago` (comer) > `antropófago`
人 + 吃 > 食人者

`filo-` (amigo) + `-sofía` (sabiduría) > `filosofía`
爱 + 智慧 > 哲学

#### 2.2.2 Compuestos populares / 通俗复合词

流行复合词由在西班牙语中作为独立单位存在的词语构成。（NGLE 11.3b）

通俗复合词由在西班牙语中作为独立单位存在的单词构成。

例如：
`lavar + platos > lavaplatos`
洗 + 盘子 > 洗碗机

`abrir + latas > abrelatas`
开 + 罐头 > 开罐器

## 3. Estructura de los compuestos / 复合词的结构

### 3.1 Compuestos coordinativos / 并列复合词

在并列复合词中，两个成分处于层级平等的关系。（NGLE 11.4a）

在并列复合词中，两个元素处于层级平等的关系中。

例如：
`agridulce` (agrio + dulce)
酸甜的（酸 + 甜）

`compraventa` (compra + venta)
买卖（买 + 卖）

### 3.2 Compuestos subordinativos / 从属复合词

在`从属复合词`中，其中一个成分修饰或补充另一个成分。(NGLE 11.4b)

在从属复合词中，一个元素修饰或补充另一个元素。

例如：
`bocacalle` (boca de la calle)
街口（街道的入口）

`coliflor` (col + flor)
花椰菜（卷心菜 + 花）

### 3.3 Compuestos con núcleo / 有核心的复合词

大多数'复合词'都有一个'核心'，该'核心'决定了'复合词'的语法属性。（NGLE 11.5a）

大多数复合词都有一个核心，决定复合词的语法属性。

例如：
`hispano + hablante > hispanohablante` (núcleo: hablante)
说西班牙语的人（核心：说话者）

`casa + quinta > casaquinta` (núcleo: casa)
带花园的房子（核心：房子）

## 4. Compuestos verbales / 动词复合词

### 4.1 Compuestos de verbo + nombre / 动词+名词复合词

动名复合词在西班牙语中非常能产。动词以陈述式现在时第三人称单数形式出现。（NGLE 11.6a）

动词加名词的复合词在西班牙语中非常能产。动词以陈述式现在时第三人称单数形式出现。

例如：
`abrelatas` (abre + latas)
开罐器（打开 + 罐头）

`paraguas` (para + aguas)
雨伞（挡住 + 水）

`sacacorchos` (saca + corchos)
开瓶器（取出 + 软木塞）

### 4.2 Compuestos de verbo + verbo / 动词+动词复合词

由两个动词构成的`compuestos`较少见。(NGLE 11.6b)

由两个动词构成的复合词较少见。

例如：
`duermevela` (duerme + vela)
半睡半醒（睡觉 + 醒着）

`ganapierde` (gana + pierde)
输赢游戏（赢 + 输）

## 5. Compuestos nominales / 名词复合词

### 5.1 Compuestos de nombre + nombre / 名词+名词复合词

名词加名词的复合词可以是`coordinativos`或`subordinativos`。(NGLE 11.7a)

名词加名词的复合词可以是并列关系或从属关系。

例如：
`café teatro` (coordinativo)
咖啡剧院（并列）

`hombre lobo` (subordinativo)
狼人（从属）

### 5.2 Compuestos de nombre + adjetivo / 名词+形容词复合词

在这些`compuestos`中，`adjetivo`修饰`nombre`。(NGLE 11.7b)

在这些复合词中，形容词修饰名词。

例如：
`aguafuerte` (agua + fuerte)
蚀刻版画（水 + 强）

`caradura` (cara + dura)
厚脸皮（脸 + 硬）

## 6. Compuestos adjetivales / 形容词复合词

### 6.1 Compuestos de adjetivo + adjetivo / 形容词+形容词复合词

形容词加形容词的复合词通常是`coordinativos`。(NGLE 11.8a)

形容词加形容词的复合词通常是并列关系。

例如：
`agridulce` (agrio + dulce)
酸甜的（酸 + 甜）

`sordomudo` (sordo + mudo)
聋哑的（聋 + 哑）

### 6.2 Compuestos de nombre + adjetivo / 名词+形容词复合词

一些形容词复合词由名词和形容词构成。(NGLE 11.8b)

一些形容词复合词由名词和形容词构成。

例如：
`pelirrojo` (pelo + rojo)
红头发的（头发 + 红色）

`bocabierto` (boca + abierto)
张着嘴的（嘴 + 张开）

## 7. Compuestos preposicionales / 介词复合词

### 7.1 Compuestos de preposición + nombre / 介词+名词复合词

由`介词`加`名词`构成的复合词在西班牙语中很常见。(NGLE 11.9a)

由介词加名词构成的复合词在西班牙语中很常见。

例如：
`sinvergüenza` (sin + vergüenza)
无耻之徒（没有 + 羞耻）

`contraportada` (contra + portada)
封底（对面 + 封面）

## 8. Compuestos con elementos cultos / 带有书面元素的复合词

### 8.1 Prefijos y sufijos cultos / 书面前缀和后缀

'elementos cultos'在构成复合词时可以用作前缀或后缀。（NGLE 11.10a）

书面元素在构成复合词时可以充当前缀或后缀。

例如：
`auto-` (por sí mismo) + `crítica` > `autocrítica`
自我 + 批评 > 自我批评

`bio-` (vida) + `logía` (estudio) > `biología`
生命 + 学 > 生物学

### 8.2 Compuestos neoclásicos / 新古典复合词

那些由源自拉丁语或希腊语的元素构成的复合词被称为`compuestos neoclásicos`。（NGLE 11.10b）

新古典复合词是指由源自拉丁语或希腊语的元素构成的复合词。

例如：
`democracia` (demos + kratos)
民主（人民 + 统治）

`televisión` (tele + visión)
电视（远 + 视觉）

## 9. Aspectos ortográficos / 正字法方面

### 9.1 Unión gráfica / 书写结合

`固有复合词`写成一个单词。(NGLE 11.11a)

固有复合词写成一个单词。

例如：
`hispanoamericano`
西班牙裔美洲人

`paraguas`
雨伞

### 9.2 Guion en los compuestos / 复合词中的连字符

在一些复合词中，使用`连字符`来分隔成分。(NGLE 11.11b)

在一些复合词中，使用连字符分隔元素。

例如：
`hispano-árabe`
西班牙-阿拉伯的

`teórico-práctico`
理论-实践的

## 10. Productividad de la composición / 复合构词法的能产性

### 10.1 Compuestos en el léxico actual / 当代词汇中的复合词

`合成法`是现代西班牙语中创造新词的一种非常能产的构词方式。（NGLE 11.12a）

在当代西班牙语中，复合构词法是一种非常能产的新词创造方式。

例如：
`ciberespacio` (ciber- + espacio)
网络空间（网络 + 空间）

`ecoturismo` (eco- + turismo)
生态旅游（生态 + 旅游）

### 10.2 Compuestos en el lenguaje técnico / 技术语言中的复合词

在技术和科学语言中，`composición` 特别能产。（NGLE 11.12b）

在技术和科学语言中，复合构词法尤其能产。

例如：
`biotecnología` (bio- + tecnología)
生物技术（生物 + 技术）

`nanotecnología` (nano- + tecnología)
纳米技术（纳米 + 技术）
