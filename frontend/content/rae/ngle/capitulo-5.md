# Capítulo 5: Derivación nominal (I): nombres deverbales / 名词派生 (I)：动词派生名词

## 5.1 Introducción / 引言

(NGLE 5.1a) 从动词派生名词是西班牙语中最能产的形态过程之一。动名词一般表示基础动词所指的动作或动作的结果。本章讨论从动词构成名词的后缀，以及所生派生词的语义和句法特性。

例如：
`El descubrimiento de América cambió la historia.`
美洲的发现改变了历史。

(NGLE 5.1b) 动名词的特点是继承了它们所派生动词的部分论元结构。因此，像 `descubrimiento` 这样的名词可以带有对应于动词 `descubrir` 的主语和宾语的补语。这一特性将它们与其他非派生自动词的名词区分开来。

例如：
`El descubrimiento de Colón de un nuevo continente`
哥伦布发现新大陆

## 5.2 Sufijos más frecuentes / 最常见的后缀

### 5.2.1 Sufijo `-ción` / 后缀 `-ción`

(NGLE 5.2a) 后缀 `-ción`（及其变体 `-ación`、`-ición`、`-oción`、`-ución`）是从动词构成动作名词的最能产后缀之一。它加在动词词干上，通常加在分词或现在时词干上。

例如：
`actuar` → `actuación`
`comer` → `comición` (forma poco usada; más común `comida`)
`construir` → `construcción`
`decidir` → `decisión`
`educar` → `educación`
`fundir` → `fusión`
`producir` → `producción`
`recibir` → `recepción`
`reducir` → `reducción`
`traducir` → `traducción`

(NGLE 5.2b) 以 `-ción` 结尾的名词主要表示动作或过程，尽管也可以表示动作的结果或效果。在许多情况下，该名词可以根据语境作两种解释。

例如：
`La construcción del puente duró dos años.` (acción)
这座桥的建设持续了两年。
`Esa construcción es muy antigua.` (resultado)
那座建筑非常古老。

(NGLE 5.2c) 有些动词对于同一个后缀存在交替形式，例如 `-ción` 和 `-miento`，有时在语义上有差异。

例如：
`aislar` → `aislamiento` / `aislación`
`enloquecer` → `enloquecimiento` / `enloquecición` (raro)

### 5.2.2 Sufijo `-miento` / 后缀 `-miento`

(NGLE 5.2d) 后缀 `-miento`（变体 `-imiento`）从动词构成动作或过程名词，尤其用于第二和第三变位动词。它加在动词的现在时词干上。

例如：
`aborrecer` → `aborrecimiento`
`conocer` → `conocimiento`
`crecer` → `crecimiento`
`establecer` → `establecimiento`
`nacer` → `nacimiento`
`padecer` → `padecimiento`
`parecer` → `parecimiento`
`permanecer` → `permanecimiento`
`reconocer` → `reconocimiento`
`sentir` → `sentimiento`
`sufrir` → `sufrimiento`
`vivir` → `vivimiento` (poco usado; más común `vida`)

(NGLE 5.2e) 与以 `-ción` 结尾的名词一样，以 `-miento` 结尾的派生词既可以表示动作，也可以表示结果。然而，它们在表达过程或状态的语境中更为常见。

例如：
`El conocimiento de la verdad fue doloroso.` (acción/estado)
了解真相是痛苦的。
`Ese edificio es un establecimiento comercial.` (resultado)
那座建筑是一家商业机构。

### 5.2.3 Sufijo `-dor` / 后缀 `-dor`

(NGLE 5.2f) 后缀 `-dor`（阴性 `-dora`）从动词构成施动者、工具或处所名词。它加在现在时词干上，通常带主题元音。

例如：
`actuar` → `actuador` / `actuadora`
`bailar` → `bailador` / `bailadora`
`cantar` → `cantador` / `cantadora`
`computar` → `computador` / `computadora`
`cultivar` → `cultivador` / `cultivadora`
`hablar` → `hablador` / `habladora`
`jugar` → `jugador` / `jugadora`
`lavar` → `lavador` / `lavadora`
`secar` → `secador` / `secadora`
`trabajar` → `trabajador` / `trabajadora`

(NGLE 5.2g) 以 `-dor` 结尾的名词可以表示：
- Agente: `cantador` (persona que canta)
- Instrumento: `secador` (aparato para secar)
- Lugar: `comedor` (lugar donde se come)

例如：
`El trabajador llegó temprano.` (agente)
工人来得很早。
`Necesito un secador de pelo nuevo.` (instrumento)
我需要一个新的吹风机。
`El comedor de la casa es amplio.` (lugar)
房子的餐厅很宽敞。

### 5.2.4 Sufijo `-nte` / 后缀 `-nte`

(NGLE 5.2h) 后缀 `-nte` 从动词构成施动者或主动分词名词。它加在现在时词干上，派生词可以用作形容词或名词。

例如：
`actuar` → `actuante`
`cantar` → `cantante`
`danzar` → `danzante`
`estudiar` → `estudiante`
`hablar` → `hablante`
`mandar` → `mandante`
`participar` → `participante`
`presidir` → `presidente`
`seguir` → `siguiente`
`viajar` → `viajante`

(NGLE 5.2i) 许多以 `-nte` 结尾的名词已词汇化为表示职业或社会角色的名词。

例如：
`El cantante actuó anoche.`
那位歌手昨晚表演了。
`Los estudiantes estudian en la biblioteca.`
学生们在图书馆学习。

### 5.2.5 Sufijo `-ncia` / 后缀 `-ncia`

(NGLE 5.2j) 后缀 `-ncia`（变体 `-encia`）从动词构成动作、性质或状态名词，尤其用于以 `-ecer`、`-ir` 或 `-er` 结尾的动词。它与后缀 `-nte` 相关。

例如：
`aparecer` → `apariencia`
`conocer` → `conocencia` (poco usado; más común `conocimiento`)
`crecer` → `crecencia`
`depender` → `dependencia`
`existir` → `existencia`
`influir` → `influencia`
`obedecer` → `obediencia`
`parecer` → `apariencia`
`permanecer` → `permanencia`
`preferir` → `preferencia`
`resistir` → `resistencia`
`tolerar` → `tolerancia`

(NGLE 5.2k) 以 `-ncia` 结尾的名词通常表示性质或状态，而非具体动作。

例如：
`La existencia del ser humano es limitada.`
人类的存在是有限的。
`La dependencia económica es un problema.`
经济依赖是一个问题。

### 5.2.6 Sufijo `-ura` / 后缀 `-ura`

(NGLE 5.2l) 后缀 `-ura` 从动词构成动作、结果或性质名词，尽管它也可以从形容词派生。它加在分词词干或现在时词干上。

例如：
`abrir` → `abertura`
`cocer` → `cocedura`
`comer` → `comidura` (poco usado; más común `comida`)
`coser` → `costura`
`criar` → `criadura`
`curar` → `curadura` (raro; más común `curación`)
`durar` → `duración` (no `dura`)
`freír` → `freidura`
`hervir` → `hervidura`
`morder` → `mordedura`
`mover` → `movidura` (poco usado)
`pintar` → `pintura`
`romper` → `rotura`
`saber` → `sabiduría` (no `sabidura`)
`torcer` → `torcedura`

(NGLE 5.2m) 以 `-ura` 结尾的名词可以表示：
- Acción: `mordedura` (acción de morder)
- Resultado: `pintura` (obra pintada)
- Cualidad: `dulzura` (cualidad de dulce, aunque este deriva de adjetivo)

例如：
`La mordedura del perro fue dolorosa.` (acción)
狗咬得很疼。
`Esa pintura es muy valiosa.` (resultado)
那幅画非常有价值。

### 5.3.7 Sufijo `-ón` / 后缀 `-ón`

(NGLE 5.3m) 后缀 `-ón`（阴性 `-ona`）从动词构成动作、击打或指大名词。它加在动词词根上。

(NGLE 5.3n) 以 `-ón` 结尾的名词通常表示击打或突然的动作，常带有指大或贬义色彩。

例如：

## 5.4 Nombres deverbales con prefijos / 带前缀的动词派生名词

(NGLE 5.4a) 一些动名词是通过在动词词干上结合前缀和后缀构成的。这些过程对于后缀 `-ción` 和 `-miento` 特别能产。

例如：
`componer` → `composición`
`descomponer` → `descomposición`
`imponer` → `imposición`
`oponer` → `oposición`
`proponer` → `proposición`
`suponer` → `suposición`

(NGLE 5.4b) 构成动名词最常见的前缀有 `a-`、`des-`、`en-`、`pre-`、`pro-`、`re-` 等。

例如：
`acercar` → `acercamiento`
`alejar` → `alejamiento`
`desarrollar` → `desarrollo`
`despegar` → `despegue`
`embellecer` → `embellecimiento`
`enloquecer` → `enloquecimiento`
`predecir` → `predicción`
`producir` → `producción`
`reconstruir` → `reconstrucción`
`reducir` → `reducción`

(NGLE 5.4c) 前缀的存在可以改变派生词相对于基础动词的意义，增加方向、重复、对立等色彩。

例如：
`El acercamiento entre los dos países fue positivo.`
两国之间的接近是积极的。
`La reconstrucción de la ciudad llevó años.`
城市的重建花了多年时间。

`plantar` → `planta`
`preguntar` → `pregunta`
`quitar` → `quita`
`saltar` → `salto`
`secar` → `seca`
`tirar` → `tiro`
`tocar` → `toca`
`tomar` → `toma`
`trabajar` → `trabajo`
`tropezar` → `tropiezo`

(NGLE 5.5b) 这些名词通常表示动作或动作的结果，在口语、体育或技术语境中特别常见。

例如：
`El ataque fue sorpresivo.`
攻击是突然的。
`La pesca es una actividad relajante.`
钓鱼是一种放松的活动。

## 5.6 Nombres deverbales con sufijos apreciativos / 带评价性后缀的动词派生名词

(NGLE 5.6a) 表义后缀（指小词、指大词、贬义词）可以加在已形成的动名词上，但也可以直接与动词词干结合构成新名词。

例如：
`bailar` → `bailecito` (diminutivo de `baile`)
`cantar` → `canturreo` (despectivo de `canto`)
`comer` → `comidita` (diminutivo de `comida`)
`golpear` → `golpecito` (diminutivo de `golpe`)
`jugar` → `jueguecito` (diminutivo de `juego`)
`mirar` → `miradita` (diminutivo de `mirada`)
`saltar` → `salto` → `saltito` (diminutivo)
`trabajar` → `trabajito` (diminutivo de `trabajo`)

(NGLE 5.6b) 表义后缀为基础名词的意义增添表达性或评价性色彩。

例如：
`Dio un saltito de alegría.`
他高兴地跳了一下。
`Hicimos un trabajito sencillo.`
我们做了一点简单的工作。

## 5.7 Nombres deverbales y estructura argumental / 动词派生名词与论元结构

(NGLE 5.7a) 动名词继承它们所派生动词的论元结构，但存在一定限制。因此，像 `descubrimiento` 这样的名词可以带有表达施事（用 `de`）和主题（用 `de`）的补语。

例如：
`El descubrimiento de Colón de América`
哥伦布发现美洲
`La construcción de los obreros del puente`
工人们建造桥梁

(NGLE 5.7b) 然而，动名词不能以相同方式表达动词的所有论元。例如，施事通常用介词 `de` 或 `por` 表达，而主题也用 `de` 表达，这可能导致歧义。

例如：
`La destrucción de la ciudad por los enemigos`
敌人对城市的摧毁
`La destrucción de los enemigos` (ambigüedad: los enemigos destruyen o son destruidos)
敌人的摧毁（歧义：敌人摧毁或被摧毁）

(NGLE 5.7c) 一些动名词可以带对应于动词论元的介词补语，如 `a`、`con`、`de`、`en`、`entre`、`para`、`por` 等。

例如：
`El amor a la patria`
对祖国的爱
`La confianza en los amigos`
对朋友的信任
`El deseo de viajar`
旅行的愿望
`La lucha por la libertad`
为自由而斗争
`El respeto a los mayores`
对长辈的尊重

(NGLE 5.7d) 动名词继承论元结构的能力因后缀和动词类型而异。以 `-ción` 和 `-miento` 结尾的名词通常比用其他后缀构成的名词更灵活。

例如：
`La aplicación de la ley por el juez`
法官对法律的适用
`El crecimiento de la economía`
经济的增长

## 5.8 Nombres deverbales y aspecto léxico / 动词派生名词与词汇体

(NGLE 5.8a) 动名词可以继承基础动词的体特征。因此，有界动词（表示有终点的动作）通常产生可解释为动作结果的名词。

例如：
`construir` (télico) → `construcción` (acción y resultado)
`La construcción del puente terminó ayer.` (acción)
`Esa construcción es moderna.` (resultado)

(NGLE 5.8b) 无界动词（表示没有内在终点的动作）通常产生表示过程或状态的名词。

例如：
`correr` (atélico) → `carrera` (proceso)
`La carrera duró una hora.`
跑步持续了一个小时。
`vivir` (atélico) → `vida` (estado)
`La vida es bella.`
生活是美好的。

(NGLE 5.8c) 一些动名词根据语境既可以解释为动作，也可以解释为结果。

例如：
`La traducción del libro fue difícil.` (acción)
翻译这本书很难。
`Esa traducción tiene errores.` (resultado)
那个翻译有错误。

## 5.9 Nombres deverbales y género / 动词派生名词与性

(NGLE 5.9a) 动名词的性在很大程度上由后缀决定。因此，后缀 `-ción`、`-ncia`、`-ura`、`-ada`、`-ida` 通常构成阴性名词，而 `-miento`、`-dor`、`-nte`、`-e`、`-o`、`-ón`、`-azo` 通常构成阳性名词。

例如：
Femeninos:
`actuación`, `existencia`, `pintura`, `caminata`, `salida`

Masculinos:
`conocimiento`, `trabajador`, `cantante`, `ataque`, `cambio`, `empujón`, `cañonazo`

(NGLE 5.9b) 然而，存在例外和变化。例如，一些以 `-dor` 结尾的名词在表示工具或机器时可以是阴性。

例如：
`La lavadora` (femenino, instrumento)
洗衣机
`El lavador` (masculino, agente)
洗涤工

(NGLE 5.9c) 以 `-nte` 结尾的名词的性可以根据所指对象变化，尽管许多已固定为阳性或阴性。

例如：
`El cantante` / `La cantante`
男歌手 / 女歌手
`El presidente` / `La presidenta`
总统（男）/ 总统（女）
