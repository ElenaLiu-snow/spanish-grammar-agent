# Capítulo 40: Modalidad: enunciados interrogativos y exclamativos / 句类：疑问句与感叹句

**Parte III: Sintaxis — Funciones y construcciones**

## 40.1 Introducción / 引言

(NGLE 40.1a) `语气`是说话人对句子内容或听话人态度的语言体现。`疑问句`和`感叹句`是两种基本的`句子语气`类型。

(NGLE 40.1b) `疑问句`表达提问或请求信息。`感叹句`表达说话人的情感反应（惊讶、赞叹、愤怒等）或对某一元素的特别强调。

例如：
`¿Has visto la película?`
你看过这部电影吗？

`¡Qué bonito es este lugar!`
这个地方多美啊！

## 40.2 Enunciados interrogativos / 疑问句

### 40.2.1 Caracterización general / 总体特征

(NGLE 40.2a) `疑问句`以其特定的语调（西班牙语中为升调）为特征，并且在许多情况下，存在诸如`qué`、`quién`、`cuál`、`cómo`、`dónde`、`cuándo`、`por qué`、`cuánto`等`疑问词`。

(NGLE 40.2b) 区分两类主要的`疑问句`：
- **Interrogativas totales**: preguntan por la verdad o falsedad de todo el contenido proposicional. Se responden con `sí` o `no`.
- **Interrogativas parciales**: preguntan por un elemento específico de la oración, introducido por un pronombre o adverbio interrogativo.

例如：
`¿Vendrás mañana?` (total)
你明天会来吗？

`¿Quién ha llamado?` (parcial)
谁打过电话？

### 40.2.2 Interrogativas totales / 一般疑问句

(NGLE 40.2c) `一般疑问句`可以不带疑问词，仅通过升调构成。在书写中用符号`¿`和`?`标记。

(NGLE 40.2d) 可能存在变体，例如：
- Interrogativas con negación: `¿No has comido?`
- Interrogativas con partículas modales: `¿Acaso no lo sabes?`

例如：
`¿Has terminado el trabajo?`
你完成工作了吗？

`¿No te parece bien?`
你不觉得好吗？

`¿Acaso piensas que miento?`
难道你认为我在说谎吗？

### 40.2.3 Interrogativas parciales / 特殊疑问句

(NGLE 40.2e) `特殊疑问句`包含一个占据句首位置的疑问词。主要的疑问词有：

- `qué`: para preguntar por cosas, conceptos o acciones
- `quién`: para personas
- `cuál`: para selección entre un conjunto
- `cómo`: para modo o manera
- `dónde`: para lugar
- `cuándo`: para tiempo
- `por qué`: para causa o razón
- `cuánto`: para cantidad

例如：
`¿Qué quieres comer?`
你想吃什么？

`¿Quién es ese hombre?`
那个人是谁？

`¿Cuál prefieres, el rojo o el azul?`
你更喜欢哪个，红色的还是蓝色的？

`¿Cómo llegaste hasta aquí?`
你是怎么到这里来的？

`¿Dónde vives?`
你住在哪里？

`¿Cuándo empieza la clase?`
课什么时候开始？

`¿Por qué no viniste ayer?`
你为什么昨天没来？

`¿Cuánto cuesta este libro?`
这本书多少钱？

### 40.2.4 Interrogativas indirectas / 间接疑问句

(NGLE 40.2f) `间接疑问句`是以非独立方式表达提问的从句。由`preguntar`、`saber`、`decir`、`ignorar`、`dudar`等动词引导。

(NGLE 40.2g) `间接一般疑问句`由连词`si`引导。`间接特殊疑问句`保留疑问词。

例如：
`No sé si vendrá.`
我不知道他是否会来。

`Pregúntale qué hora es.`
问问他几点了。

`Dime dónde está la estación.`
告诉我车站在哪里。

`Ignoro por qué se enfadó.`
我不知道他为什么生气了。

### 40.2.5 Interrogativas retóricas / 修辞疑问句

(NGLE 40.2h) `反问句`不是为了获取信息，而是表达强调的肯定或情感反应。它们通常隐含答案。

(NGLE 40.2i) 可以是`一般反问句`或`特殊反问句`。在许多情况下，它们相当于否定或绝对肯定。

例如：
`¿Quién puede negar la evidencia?` (= Nadie puede negarla)
谁能否认证据呢？

`¿Acaso no es hermoso este paisaje?` (= Es hermoso)
难道这风景不美吗？

`¿Qué más se puede pedir?` (= No se puede pedir más)
还能要求什么呢？

## 40.3 Enunciados exclamativos / 感叹句

### 40.3.1 Caracterización general / 总体特征

(NGLE 40.3a) `感叹句`表达说话人的情感态度（惊讶、赞叹、喜悦、痛苦、愤怒等）。以其特定语调（降升调）和存在诸如`qué`、`quién`、`cómo`、`dónde`、`cuándo`、`cuánto`等`感叹词`为特征。

(NGLE 40.3b) 在书写中用符号`¡`和`!`标记。

例如：
`¡Qué sorpresa verte aquí!`
在这里见到你真是太惊喜了！

`¡Cómo has crecido!`
你长这么大了！

### 40.3.2 Elementos exclamativos / 感叹词

(NGLE 40.3c) 主要的`感叹词`有：

- `qué`: el más frecuente, puede modificar sustantivos, adjetivos o adverbios
- `quién`: para personas, a menudo con valor de sorpresa
- `cómo`: para modo o intensidad
- `dónde`: para lugar, con valor enfático
- `cuándo`: para tiempo, con valor enfático
- `cuánto`: para cantidad o intensidad

(NGLE 40.3d) `Qué`可出现在如下结构中：
- `¡Qué + sustantivo!`: `¡Qué casa!`
- `¡Qué + adjetivo!`: `¡Qué bonito!`
- `¡Qué + adverbio!`: `¡Qué bien!`
- `¡Qué + sustantivo + tan/más + adjetivo!`: `¡Qué día tan maravilloso!`

例如：
`¡Qué casa tan elegante!`
多么优雅的房子啊！

`¡Qué bonito es este cuadro!`
这幅画多美啊！

`¡Qué bien cocinas!`
你做饭做得真好！

`¡Qué día más horrible!`
多么糟糕的一天啊！

`¡Quién lo hubiera dicho!`
谁会想到呢！

`¡Cómo llueve!`
雨下得多大啊！

`¡Dónde iremos a parar!`
我们会落到什么地步啊！

`¡Cuándo se habrá visto algo así!`
什么时候见过这种事啊！

`¡Cuánto lo siento!`
我多么遗憾啊！

### 40.3.3 Exclamativas de grado / 程度感叹句

(NGLE 40.3e) `感叹句`可通过如下结构表达程度或强度：
- `¡Qué + tan/más + adjetivo/adverbio!`
- `¡Vaya + sustantivo!`
- `¡Cómo + verbo!`

(NGLE 40.3f) `Vaya`用作感叹词，常具有讽刺或强调意味。

例如：
`¡Qué tan lejos está!`
它多么远啊！

`¡Vaya lío que has armado!`
你惹了多大的麻烦啊！

`¡Cómo corre ese niño!`
那个孩子跑得多快啊！

### 40.3.4 Exclamativas indirectas / 间接感叹句

(NGLE 40.3g) `间接感叹句`是以非独立方式表达感叹的从句。由`decir`、`exclamar`、`pensar`、`ver`等动词引导。

(NGLE 40.3h) 保留特有的感叹词。

例如：
`No te imaginas qué difícil fue.`
你想象不到有多难。

`Mira qué bonito es.`
看它多美。

`Dime qué sorpresa tan grande.`
告诉我多么大的惊喜。

## 40.4 Relación entre interrogación y exclamación / 疑问与感叹的关系

(NGLE 40.4a) 疑问和感叹之间存在密切关系。许多句子可根据语境和语调被理解为疑问句或感叹句。

(NGLE 40.4b) 有些句子可能在两种解释之间产生歧义：

例如：
`¡Qué bien has hecho!` (exclamativa)
你做得真好！

`¿Qué bien has hecho?` (interrogativa)
你做了什么好事？

(NGLE 40.4c) `反问句`具有接近`感叹句`的价值，因为它们表达说话人的情感态度，而非信息请求。

例如：
`¿Quién puede creer eso?` (retórica, casi exclamativa)
谁能相信那个呢？

## 40.5 Signos de puntuación / 标点符号

(NGLE 40.5a) 在西班牙语中，疑问句和感叹句用相应的起始符号（`¿` `¡`）和结束符号（`?` `!`）书写。

(NGLE 40.5b) 与英语或法语等其他语言不同，西班牙语中起始符号是强制性的。

(NGLE 40.5c) 可结合使用来表达混合的怀疑或惊讶：
- `¿¡` o `¡¿` para indicar sorpresa interrogativa

例如：
`¿¡Cómo es posible!?`
怎么可能？！

`¡¿Qué has hecho?!`
你做了什么？！

## 40.6 Resumen / 总结

(NGLE 40.6a) `疑问句`和`感叹句`是西班牙语中两种基本的`句子语气`类型。

(NGLE 40.6b) `疑问句`分为`一般疑问句`（是/否回答）和`特殊疑问句`（带疑问词）。可以是直接或间接形式，也包括`反问句`。

(NGLE 40.6c) `感叹句`通过`qué`、`cómo`、`cuánto`等元素表达情感。可以是直接或间接形式。

(NGLE 40.6d) 两类句子共享许多词汇元素和句法结构，主要区别在于语调与交际意图。

(NGLE 40.6e) 起始符号（`¿` `¡`）是西班牙语的特点，在书写中是强制性的。
