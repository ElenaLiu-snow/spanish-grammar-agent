# Capítulo 6: Derivación nominal (II): nombres deadjetivales y denominales / 名词派生 (II)：形容词和名词派生名词

## 1. Introducción / 引言

(NGLE 6.1a) 本章继续研究名词派生，重点讨论由形容词形成的名词（形源名词）和由其他名词形成的名词（名源名词）。分析了最高产的词缀及其意义。

## 2. Nombres deadjetivales / 形容词派生名词

### 2.1 Sufijos que forman nombres de cualidad / 构成性质名词的后缀

(NGLE 6.2a) 由形容词形成性质名词最典型的词缀包括`-idad`、`-eza`、`-ura`、`-ía`、`-ez`、`-icia`、`-itud`、`-or`、`-ancia`、`-encia`等。

#### 2.1.1 Sufijo `-idad` / 后缀 `-idad`

(NGLE 6.2b) 词缀`-idad`是从形容词形成抽象性质名词最高产的词缀。它添加在形容词词基上，有时伴有语音变化。

例如：
`bello` → `belleza` (no *belidad)
`claro` → `claridad`
`difícil` → `dificultad`
`feliz` → `felicidad`
`nuevo` → `novedad`
`simple` → `simplicidad`

(NGLE 6.2c) 许多以`-ble`结尾的形容词用`-bilidad`构成名词。

例如：
`amable` → `amabilidad`
`posible` → `posibilidad`
`responsable` → `responsabilidad`
`visible` → `visibilidad`

(NGLE 6.2d) 以`-io`结尾的形容词通常将`o`变为`-idad`。

例如：
`serio` → `seriedad`
`vario` → `variedad`

#### 2.1.2 Sufijo `-eza` / 后缀 `-eza`

(NGLE 6.3a) 词缀`-eza`从形容词形成性质名词，常用于表达身体或道德品质的形容词。

例如：
`bello` → `belleza`
`grande` → `grandeza`
`limpio` → `limpieza`
`triste` → `tristeza`
`dulce` → `dulzura` (con sufijo `-ura`)

(NGLE 6.3b) 一些形容词既用`-eza`也用`-idad`构成名词，有时用法有差异。

例如：
`agudo` → `agudeza` / `agudeza` (más común)
`áspero` → `aspereza` / `asperidad`
`fino` → `fineza` / `finura`

#### 2.1.3 Sufijo `-ura` / 后缀 `-ura`

(NGLE 6.4a) 词缀`-ura`从形容词形成性质名词，尤其指称物理或感官属性的形容词。

例如：
`alto` → `altura`
`ancho` → `anchura`
`blanco` → `blancura`
`dulce` → `dulzura`
`fresco` → `frescura`
`negro` → `negrura`
`tierno` → `ternura`

(NGLE 6.4b) 一些形容词用`-ura`构成名词，也能用其他词缀。

例如：
`amargo` → `amargura` / `amargor`
`caliente` → `calentura` / `calor`

#### 2.1.4 Sufijo `-ía` / 后缀 `-ía`

(NGLE 6.5a) 词缀`-ía`从形容词形成性质名词，常用于表达道德或社会品质的形容词。

例如：
`alegre` → `alegría`
`cobarde` → `cobardía`
`falso` → `falsía`
`valiente` → `valentía`
`villano` → `villanía`

(NGLE 6.5b) 也从以`-ario`结尾的形容词构成名词。

例如：
`contrario` → `contrariedad` (con `-edad`)
`necesario` → `necesidad` (con `-idad`)

#### 2.1.5 Sufijo `-ez` / 后缀 `-ez`

(NGLE 6.6a) 词缀`-ez`从形容词形成性质名词，常用于表达身体或性格品质的形容词。

例如：
`altivo` → `altivez`
`brusco` → `brusquedad` (con `-edad`)
`cortés` → `cortesía` (con `-ía`)
`delgado` → `delgadez`
`esbelto` → `esbeltez`
`fino` → `fineza` (con `-eza`)
`maduro` → `madurez`
`obscuro` → `obscuridad` (con `-idad`)
`tonto` → `tontería` (con `-ía`)
`veloz` → `velocidad` (con `-idad`)

(NGLE 6.6b) 一些形容词用`-ez`构成名词，也能用其他词缀。

例如：
`agudo` → `agudeza` / `agudez`
`áspero` → `aspereza` / `asperez`
`fino` → `fineza` / `finez`

#### 2.1.6 Sufijo `-icia` / 后缀 `-icia`

(NGLE 6.7a) 词缀`-icia`从形容词形成性质名词，常用于以`-icio`或`-izo`结尾的形容词。

例如：
`avaricia` (de `avaro`)
`codicia` (de `codicioso`)
`justicia` (de `justo`)
`malicia` (de `malo`)
`pericia` (de `perito`)

#### 2.1.7 Sufijo `-itud` / 后缀 `-itud`

(NGLE 6.8a) 词缀`-itud`从形容词形成性质名词，常用于表达状态或条件的形容词。

例如：
`alto` → `altitud`
`amplio` → `amplitud`
`largo` → `longitud`
`recto` → `rectitud`
`solo` → `soledad` (con `-edad`)
`tranquilo` → `tranquilidad` (con `-idad`)

#### 2.1.8 Sufijo `-or` / 后缀 `-or`

(NGLE 6.9a) 词缀`-or`从形容词形成性质名词，常用于表达物理或感官属性的形容词。

例如：
`amargo` → `amargor`
`caliente` → `calor`
`dulce` → `dulzor`
`fresco` → `frescor`
`negro` → `negror`
`rojo` → `rojez` (con `-ez`)
`verde` → `verdor`

(NGLE 6.9b) 一些形容词用`-or`构成名词，也能用其他词缀。

例如：
`amargo` → `amargor` / `amargura`
`dulce` → `dulzor` / `dulzura`

#### 2.1.9 Sufijos `-ancia` y `-encia` / 后缀 `-ancia` 和 `-encia`

(NGLE 6.10a) 词缀`-ancia`和`-encia`从以`-ante`或`-ente`结尾的形容词形成性质名词。

例如：
`abundante` → `abundancia`
`elegante` → `elegancia`
`importante` → `importancia`
`inteligente` → `inteligencia`
`paciente` → `paciencia`
`persistente` → `persistencia`

(NGLE 6.10b) 也从不以`-ante`或`-ente`结尾的形容词构成。

例如：
`ausente` → `ausencia`
`presente` → `presencia`
`violento` → `violencia`

## 3. Nombres denominales / 名词派生名词

### 3.1 Sufijos que forman nombres de persona / 构成人称名词的后缀

(NGLE 6.16a) 名源名词从其他名词形成人称名词，表示职业、手艺、意识形态、籍贯等。

#### 3.1.1 Sufijo `-ista` / 后缀 `-ista`

(NGLE 6.16b) 词缀`-ista`从名词形成人称名词，表示某种学说的追随者、职业或活动。

例如：
`arte` → `artista`
`ciencia` → `científico` (con `-ífico`)
`deporte` → `deportista`
`economía` → `economista`
`fútbol` → `futbolista`

`piano` → `pianista`
`violín` → `violinista`

#### 3.1.2 Sufijo `-ero` / 后缀 `-ero`

(NGLE 6.17a) 词缀`-ero`从名词形成人称名词，表示职业、手艺或活动。

例如：
`barco` → `barquero`
`carnicería` → `carnicero`
`coche` → `cochero`

`herrería` → `herrero`

#### 3.1.3 Sufijo `-or` / 后缀 `-or`

(NGLE 6.18a) 词缀`-or`从名词形成人称名词，常与动词有关。

例如：

#### 3.1.4 Sufijo `-ario` / 后缀 `-ario`

`catedrático` (de `cátedra`)
`comisario` (de `comisión`)
`secretario` (de `secreto`)

#### 3.1.5 Sufijo `-ano` / 后缀 `-ano`

例如：

`América` → `americano`

#### 3.1.7 Sufijo `-és` / 后缀 `-és`

(NGLE 6.22a) 词缀`-és`从地名形成人称名词，表示籍贯。

#### 3.1.8 Sufijo `-eño` / 后缀 `-eño`

## 4. Sufijos diminutivos y aumentativos / 指小和指大后缀

### 4.1 Sufijos diminutivos / 指小后缀

(NGLE 6.43a) 指小词缀形成表示小尺寸或情感色彩的名词。最常见的是`-ito`、`-illo`、`-ico`、`-ín`、`-uelo`。

#### 4.1.1 Sufijo `-ito` / 后缀 `-ito`

(NGLE 6.43b) 词缀`-ito`是现代西班牙语中最常见的指小词缀。

例如：
`casa` → `casita`
`libro` → `librito`
`mesa` → `mesita`
`niño` → `niñito`
`perro` → `perrito`

#### 4.1.2 Sufijo `-illo` / 后缀 `-illo`

(NGLE 6.44a) 词缀`-illo`具有指小意义，有时带有贬义色彩。

例如：
`casa` → `casilla`
`libro` → `librillo`
`mesa` → `mesilla`
`niño` → `niñillo`
`perro` → `perrillo`

#### 4.1.3 Sufijo `-ico` / 后缀 `-ico`

(NGLE 6.45a) 词缀`-ico`在某些地区如安达卢西亚和美洲常见。

例如：
`casa` → `casica`
`libro` → `librico`
`mesa` → `mesica`
`niño` → `niñico`
`perro` → `perrico`

#### 4.1.4 Sufijo `-ín` / 后缀 `-ín`

(NGLE 6.46a) 词缀`-ín`构成指小词，常带有情感色彩。

例如：
`casa` → `casín`
`libro` → `librín`
`mesa` → `mesín`

(NGLE 6.57a) 西班牙语的名词派生是一个非常高产的过程，可以从形容词和名词创造新名词。本章介绍的词缀是最常见的，但也存在其他较低产的词缀。词缀的选择取决于词基、所需意义和语言传统等因素。
