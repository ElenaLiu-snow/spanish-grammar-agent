# Capítulo 9: Derivación apreciativa / 评价性派生

## 9.1 Introducción / 引言

(NGLE 9.1a) 表意派生是一种形态派生，为词根添加表达或评价色彩，而不改变其语法类别或基本概念意义。这些色彩可以是大小、喜爱、轻蔑、讽刺等。

(NGLE 9.1b) 表意后缀传统上分为三类：`指小词`、`指大词`和`贬义词`。然而，这种分类并不详尽，因为许多后缀可以根据语境表达不止一种意义。

例如：
`El perro pequeño es muy bonito.`
那只小狗很漂亮。

## 9.2 Sufijos diminutivos / 指小后缀

### 9.2.1 Características generales / 一般特征

(NGLE 9.2a) 西班牙语中最常见的`指小词`后缀有`-ito/-ita`、`-illo/-illa`、`-ico/-ica`、`-ín/-ina`、`-ino/-ina`、`-uelo/-uela`、`-ete/-eta`、`-ico/-ica`、`-iño/-iña`。它们的分布因西语世界地区而异。

(NGLE 9.2b) 后缀`-ito/-ita`是当今西班牙语中使用最广泛且最具能产性的。它加在名词和形容词上，偶尔也加在副词上。

例如：
`casa` → `casita`
`perro` → `perrito`
`ahora` → `ahorita`

(NGLE 9.2c) `指小词`不仅表示小的尺寸，还可以表达喜爱、礼貌、缓和或讽刺。

例如：
`Un momento, por favor.` → `Un momentito, por favor.`
请稍等片刻。

### 9.2.2 Formación de los diminutivos / 指小词的形成

(NGLE 9.2d) `指小词`的构成遵循音系和形态规则。通常，在添加后缀之前要删除词根的末尾元音，但某些情况除外。

例如：
`mesa` → `mesita`
`flor` → `florecita`
`café` → `cafecito`

(NGLE 9.2e) 一些词根会发生语音变化，如双元音化或辅音丢失。

例如：
`piedra` → `piedrecita`
`huevo` → `huevito`

### 9.2.3 Diminutivos con valor afectivo / 带有情感色彩的指小词

(NGLE 9.2f) 表喜爱的`指小词`在口语和儿童语言中非常常见。它们可以表达亲切、温柔或熟悉。

例如：
`mamá` → `mamacita`
`papá` → `papacito`
`abuelo` → `abuelito`

(NGLE 9.2g) 它们也用于缓和命令或请求，或表示礼貌。

例如：
`Siéntate.` → `Siéntate un ratito.`
你坐一会儿吧。

## 9.3 Sufijos aumentativos / 指大后缀

### 9.3.1 Características generales / 一般特征

(NGLE 9.3a) 最常见的`指大词`后缀有`-ón/-ona`、`-ote/-ota`、`-azo/-aza`、`-acho/-acha`、`-ucho/-ucha`。它们表示大的尺寸，但也可以表达强度、夸张或负面评价。

例如：
`casa` → `casona`
`perro` → `perrote`
`golpe` → `golpazo`

(NGLE 9.3b) 后缀`-ón/-ona`是最具能产性的。除了表示大小，它还可以表示打击或暴力动作。

例如：
`portazo` (golpe de puerta)
`manotazo` (golpe con la mano)

### 9.3.2 Aumentativos con valor despectivo / 带有贬义色彩的指大词

(NGLE 9.3c) 许多`指大词`带有贬义或轻蔑的含义，尤其是在指人时。

例如：
`mujer` → `mujerona` (mujer grande y tosca)
`hombre` → `hombrón` (hombre grande y fuerte, a veces con admiración)

(NGLE 9.3d) 后缀`-ote/-ota`通常带有轻蔑或嘲弄的色彩。

例如：
`amigo` → `amigote` (amigo con familiaridad, a veces despectivo)
`casa` → `casota` (casa grande y fea)

## 9.4 Sufijos despectivos / 贬义后缀

### 9.4.1 Características generales / 一般特征

(NGLE 9.4a) `贬义词`后缀表达轻蔑、厌恶、嘲弄或讽刺。最常见的有`-aco/-aca`、`-acho/-acha`、`-ajo/-aja`、`-ejo/-eja`、`-ucho/-ucha`、`-uzo/-uza`、`-astro/-astra`、`-orrio/-orria`。

例如：
`casa` → `casucha`
`perro` → `perracho`
`libro` → `libraco`

(NGLE 9.4b) 后缀`-ucho/-ucha`具有很高的能产性，可以应用于名词和形容词。

例如：
`feo` → `feúcho`
`pobre` → `pobrucho`

### 9.4.2 Despectivos con valor atenuante / 带有弱化色彩的贬义词

(NGLE 9.4c) 一些`贬义词`可能具有缓和或幽默的作用，尤其是在家庭语境中。

例如：
`médico` → `medicucho` (médico de poca categoría)
`poeta` → `poetastro` (poeta malo)

## 9.5 Sufijos con valor ponderativo / 带有强调价值的后缀

### 9.5.1 Características generales / 一般特征

(NGLE 9.5a) `强调词`后缀表达强度、夸张或赞赏。最常见的有`-ísimo/-ísima`、`-érrimo/-érrima`（`最高级`）以及在某些用法中的`-azo/-aza`。

例如：
`grande` → `grandísimo`
`pobre` → `pobrísimo`
`libro` → `librazo` (libro grande o importante)

(NGLE 9.5b) 后缀`-azo/-aza`在表示打击或冲击时可能具有强调意义，但也用于表达赞赏。

例如：
`temporada` → `temporadaza` (temporada muy buena)
`partido` → `partidazo` (partido excelente)

## 9.6 Sufijos con valor iterativo o frecuentativo / 带有重复或频繁价值的后缀

(NGLE 9.6a) 一些`表意后缀`可以表示重复或频率，尤其是在派生动词中。

例如：
`golpear` → `golpetear` (golpear repetidamente)
`besar` → `besuquear` (besar mucho)

(NGLE 9.6b) 这些后缀通常与动词词根结合，以表达重复或强度较小的动作。

例如：
`llover` → `lloviznar` (llover ligeramente)
`comer` → `comisquear` (comer poco y a menudo)

## 9.7 Interfijos en la derivación apreciativa / 评价性派生中的中缀

(NGLE 9.7a) 在构成`表意派生词`时，有时会出现`中缀`（词根和后缀之间的非重读成分），以便于发音或避免语音变化。

例如：
`flor` → `florecita` (interfijo `-ec-`)
`café` → `cafecito` (interfijo `-ec-`)
`pan` → `panecillo` (interfijo `-ec-`)

(NGLE 9.7b) 最常见的`中缀`有`-ec-`、`-et-`、`-ic-`、`-ill-`、`-it-`。它们的存在取决于词根的语音结构。

例如：
`pez` → `pececito`
`sol` → `solecito`
`árbol` → `arbolito` (sin interfijo)

## 9.8 Derivación apreciativa y cambio de categoría / 评价性派生与词类变化

(NGLE 9.8a) 尽管`表意派生`通常不改变词根的语法类别，但存在例外。一些后缀可以将名词转化为形容词，反之亦然。

例如：
`niño` (sustantivo) → `niñato` (adjetivo despectivo)
`viejo` (adjetivo) → `vejete` (sustantivo despectivo)

(NGLE 9.8b) `表意后缀`也可以应用于副词、感叹词和其他词类。

例如：
`cerca` → `cerquita`
`lejos` → `lejitos`
`adiós` → `adiosito`

## 9.9 Variación geográfica / 地理变异

(NGLE 9.9a) `表意后缀`的使用在西语世界不同地区之间差异很大。

(NGLE 9.9b) 在西班牙，最常见的后缀是`-ito/-ita`和`-illo/-illa`。在美洲，`-ito/-ita`占主导地位，尽管在某些地区使用`-ico/-ica`（哥伦比亚、委内瑞拉、古巴）或`-iño/-iña`（加利西亚、阿根廷）。

例如：
`chico` → `chiquito` (general)
`chico` → `chiquillo` (España)
`chico` → `chiquico` (Colombia)
`chico` → `chiquiño` (Argentina)

(NGLE 9.9c) 在加勒比地区和中美洲，后缀`-ón/-ona`在指大和贬义用法上具有很强的能产性。

例如：
`mujer` → `mujerona` (Caribe)
`hombre` → `hombrón` (Centroamérica)

## 9.10 Aspectos semánticos y pragmáticos / 语义与语用方面

(NGLE 9.10a) `表意后缀`具有丰富的语义和语用价值。其解释取决于语境、语调和说话者之间的关系。

(NGLE 9.10b) 同一个后缀可以根据语境表达相反的意义。例如，`-ito/-ita`可以表示喜爱或讽刺。

例如：
`¡Qué casita más bonita!` (cariño)
`¡Vaya casita que te has comprado!` (ironía, si la casa es pequeña o fea)

(NGLE 9.10c) `表意后缀`在口语、儿童语言和情感语言中非常常见。它们也用于广告和媒体中，以营造与公众的亲近感。

例如：
`Compra nuestro jaboncito.`
买我们的小肥皂吧。

## 9.11 Restricciones en la derivación apreciativa / 评价性派生的限制

(NGLE 9.11a) 并非所有词根都能接受`表意后缀`。存在语义、语音和形态限制。

(NGLE 9.11b) 表示抽象实体或大尺寸事物的词根通常拒绝`指小词`。

例如：
`*eternidad` → `*eternidadita`
`*universo` → `*universito`

(NGLE 9.11c) 具有某些语音结尾的词根可能需要`中缀`或形式变化。

例如：
`tren` → `trenecito` (con interfijo)
`ley` → `leyecita` (con interfijo)

(NGLE 9.11d) 一些`表意后缀`比其他更具备能产性。`-ito/-ita`是最具能产性的，而`-astro/-astra`能产性较低，仅限于某些词根。

例如：
`poetastro` (poeta malo)
`medicastro` (médico malo)

## 9.12 Conclusión / 结论

(NGLE 9.12a) `表意派生`是西班牙语中非常能产的形态机制，允许表达广泛的表现和评价色彩。

(NGLE 9.12b) `表意后缀`分为`指小词`、`指大词`、`贬义词`和`强调词`，尽管其语义价值复杂且取决于语境。

(NGLE 9.12c) 对`表意派生`的研究揭示了西班牙语丰富的表达能力及其适应不同语域和交际情境的能力。

例如：
`El uso de diminutivos y aumentativos es una característica distintiva del español coloquial.`
指小词和指大词的使用是西班牙语口语的一个显著特征。
