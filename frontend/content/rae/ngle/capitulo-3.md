# Capítulo 3: El número / 数

## 1. Introducción / 引言

(NGLE 3.1a) `数`是区分单数形式和复数形式的语法属性。单数指称一个个体或一个单位，而复数指称多个个体或多个单位。

例如：
`casa / casas`
房子 / 多所房子

`árbol / árboles`
树 / 多棵树

(NGLE 3.1b) `数`影响名词、形容词、限定词、代词和动词。在名词和形容词中，`数`通过添加在词干上的屈折词缀来体现。

## 2. Formación del plural / 复数的构成

### 2.1 Reglas generales / 一般规则

(NGLE 3.2a) 西班牙语中构成复数的通用规则是：以非重读元音或重读`-e`结尾的词加`-s`，以辅音或除`-e`以外的重读元音结尾的词加`-es`。

例如：
`casa / casas`
房子 / 多所房子

`libro / libros`
书 / 多本书

`hombre / hombres`
男人 / 男人们

`sofá / sofás`
沙发 / 多个沙发

`rubí / rubíes`
红宝石 / 多颗红宝石

(NGLE 3.2b) 以元音加`-y`结尾的词构成复数时加`-es`，尽管在正字法上`y`被视为辅音。

例如：
`ley / leyes`
法律 / 多项法律

`rey / reyes`
国王 / 多位国王

`buey / bueyes`
牛 / 多头牛

### 2.2 Palabras terminadas en vocal tónica / 以重读元音结尾的词

(NGLE 3.3a) 以`-á`、`-é`、`-ó`结尾的重读词尾词构成复数时加`-s`。

例如：
`bajá / bajás`
帕夏 / 多位帕夏

`café / cafés`
咖啡 / 多杯咖啡

`dominó / dominós`
多米诺骨牌 / 多副多米诺骨牌

(NGLE 3.3b) 以`-í`和`-ú`结尾的重读词尾词有两种复数形式：最常见的是加`-es`，但也使用加`-s`的形式，尤其在美洲西班牙语中。

例如：
`alhelí / alhelíes` (también `alhelís`)
紫罗兰 / 多株紫罗兰

`tabú / tabúes` (también `tabús`)
禁忌 / 多项禁忌

`rubí / rubíes` (también `rubís`)
红宝石 / 多颗红宝石

### 2.3 Palabras terminadas en consonante / 以辅音结尾的词

(NGLE 3.4a) 以辅音（`-s`或`-x`除外）结尾的词构成复数时加`-es`。

例如：
`árbol / árboles`
树 / 多棵树

`pared / paredes`
墙 / 多面墙

`reloj / relojes`
钟 / 多个钟

`color / colores`
颜色 / 多种颜色

(NGLE 3.4b) 以`-s`或`-x`结尾且为重读词尾或单音节的词构成复数时加`-es`。以`-s`或`-x`结尾的正式词或超重读音词在复数中不变。

例如：
`mes / meses`
月 / 多个月

`país / países`
国家 / 多个国家

`compás / compases`
圆规 / 多个圆规

`lunes / lunes` (不变)
星期一 / 多个星期一

`crisis / crisis` (不变)
危机 / 多次危机

`tórax / tórax` (不变)
胸廓 / 多个胸廓

### 2.4 Palabras terminadas en `-z` / 以 `-z` 结尾的词

(NGLE 3.5a) 以`-z`结尾的词在加`-es`前将`z`改为`c`。

例如：
`luz / luces`
光 / 多束光

`voz / voces`
声音 / 多种声音

`feliz / felices`
幸福的 / 幸福的（复数）

`actriz / actrices`
女演员 / 多位女演员

`pez / peces`
鱼 / 多条鱼

## 3. Plural de los compuestos / 复合词的复数

(NGLE 3.6a) 在连写形式的复合词中，复数通常构成在第二个成分上。

例如：
`hispanoamericano / hispanoamericanos`
西班牙裔美洲人 / 西班牙裔美洲人们

`bocacalle / bocacalles`
街口 / 多个街口

`mediodía / mediodías`
正午 / 多个正午

(NGLE 3.6b) 在由连字符分隔的两个词构成的复合词中，如果两个成分都是名词，则复数影响两者；如果第一个成分不变，则只影响第二个。

例如：
`coche-cama / coches-cama`
卧铺车 / 多辆卧铺车

`sofá-cama / sofás-cama`
沙发床 / 多张沙发床

`hijo político / hijos políticos`
女婿 / 多个女婿

(NGLE 3.6c) 在由多个词构成的固定名词短语中，复数通常影响短语的核心成分。

例如：
`ojo de buey / ojos de buey`
舷窗 / 多个舷窗

`carta de amor / cartas de amor`
情书 / 多封情书

`hombre rana / hombres rana`
蛙人 / 多位蛙人

## 4. Plural de los extranjerismos / 外来词的复数

(NGLE 3.7a) 被西班牙语同化的外来词遵循复数构成的一般规则。

例如：
`chalé / chalés`
别墅 / 多栋别墅

`bistec / bistecs`
牛排 / 多块牛排

`club / clubes`
俱乐部 / 多个俱乐部

(NGLE 3.7b) 未被同化的外来词通常保留其原始复数形式，但也可以使用西班牙语的规则。

例如：
`el rock / los rocks` (también `los roques`)
摇滚乐 / 多种摇滚乐

`el jazz / los jazz` (不变)
爵士乐 / 多种爵士乐

`el software / los software` (不变)
软件 / 多个软件

`el ballet / los ballets`
芭蕾舞 / 多场芭蕾舞

## 5. Plural de los nombres propios / 专有名词的复数

(NGLE 3.8a) 人名使用复数形式，当指代多个拥有相同名字的人时，或当用作呼语时。

例如：
`los Juanes`
多个胡安

`las Marías`
多个玛丽亚

`los Garcías`
多个加西亚

(NGLE 3.8b) 地名可以使用复数形式，当指代以该名称命名的区域或国家时，或当指代多个地理实体时。

例如：
`las Américas`
美洲各国

`los Andes`
安第斯山脉

`las Filipinas`
菲律宾群岛

## 6. Sustantivos que solo tienen singular o solo plural / 只有单数或只有复数的名词

### 6.1 Singularia tantum / 单数名词

(NGLE 3.9a) 仅用于单数的名词称为"单数伴随词"。它们指代不被视为复数的事物。

例如：
`la salud`
健康

`la sed`
口渴

`el caos`
混乱

`el oeste`
西方

`la sangre`
血

`el norte`
北方

(NGLE 3.9b) 一些通常为单数伴随词的名词可以在特殊语境或含义变化时用于复数。

例如：
`las aguas` (en lugar de `el agua`)
水域

`los aires` (en lugar de `el aire`)
空气（复数形式表示特定含义）

`las sangres` (en lugar de `la sangre`)
血液（复数形式表示不同血型或种族）

### 6.2 Pluralia tantum / 复数名词

(NGLE 3.10a) 仅用于复数的名词称为"复数伴随词"。它们指代被视为复数或由多个部分构成的事物。

例如：
`las afueras`
郊区

`las gafas`
眼镜

`las tijeras`
剪刀

`los víveres`
食物

`los comestibles`
食品

`los alrededores`
周围

(NGLE 3.10b) 一些复数伴随词可以在含义变化时用于单数。

例如：
`la gafa` (en lugar de `las gafas`)
眼镜片

`la tijera` (en lugar de `las tijeras`)
剪刀（单数形式指一把剪刀）

`el viver` (en lugar de `los víveres`)
食物（单数形式较少用）

## 7. El número en los adjetivos / 形容词的数

形容词与所修饰的名词在数上保持一致。形容词的复数构成遵循与名词相同的规则。

例如：
`casa blanca / casas blancas`
白房子 / 多所白房子

`hombre alto / hombres altos`
高个子男人 / 高个子男人们

`mujer feliz / mujeres felices`
幸福的女人 / 幸福的女人们

`día gris / días grises`
灰色的一天 / 多个灰色的日子

以`-z`结尾的形容词在复数中将`z`变为`c`，与名词相同。

例如：
`capaz / capaces`
有能力的 / 有能力的（复数）

`feroz / feroces`
凶猛的 / 凶猛的（复数）

`veloz / veloces`
快速的 / 快速的（复数）

## 8. El número en los determinantes y pronombres / 限定词和代词中的数

限定词和代词也有数的变化，必须与它们伴随或所指的名词一致。

例如：
`este libro / estos libros`
这本书 / 这些书

`esa casa / esas casas`
那所房子 / 那些房子

`mi amigo / mis amigos`
我的朋友 / 我的朋友们

`tu hermana / tus hermanas`
你的姐妹 / 你的姐妹们

一些限定词和代词有特殊的复数形式。

例如：
`un / unos`
一个 / 一些

`ningún / ningunos`
没有一个 / 没有一些

`alguien / algunos`
某人 / 某些人

`nadie / nadies` (poco usado)
没有人 / 没有人（复数形式很少用）

## 9. El número en los verbos / 动词中的数

动词与句子的主语在数上保持一致。动词的数屈折通过动词词尾体现。

例如：
`El niño juega. / Los niños juegan.`
男孩在玩。 / 男孩们在玩。

`La mujer canta. / Las mujeres cantan.`
女人在唱歌。 / 女人们在唱歌。

`Tú estudias. / Vosotros estudiáis.`
你学习。 / 你们学习。

在复合动词形式中，当与动词`ser`一起使用时，分词在数上与主语一致以构成被动语态，但当与`haber`一起使用时则不一致以构成复合时态。

例如：
`La carta fue escrita. / Las cartas fueron escritas.`
信被写了。 / 多封信被写了。

`Él ha escrito la carta. / Ellos han escrito las cartas.`
他写了信。 / 他们写了多封信。

## 10. Casos especiales de concordancia de número / 数的一致性的特殊情况

当主语由多个并列名词构成时，动词用复数，即使每个名词都是单数。

例如：
`Juan y María estudian.`
胡安和玛丽亚在学习。

`El libro y el cuaderno están sobre la mesa.`
书和笔记本在桌子上。

当主语是单数集合名词时，动词可根据上下文和用法使用单数或复数。

例如：
`La gente canta. / La gente cantan.` (menos frecuente)
人们在唱歌。

`El público aplaude. / El público aplauden.` (menos frecuente)
观众在鼓掌。

`La multitud avanza. / La multitud avanzan.` (menos frecuente)
人群在前进。

当主语是带有量化词如`la mayoría`、`el resto`、`la mitad`的名词短语时，动词可与核心词在数上保持单数一致，或与补语在数上保持复数一致。

例如：
`La mayoría de los estudiantes aprueba. / La mayoría de los estudiantes aprueban.`
大多数学生通过了。

`El resto de los libros está en la biblioteca. / El resto de los libros están en la biblioteca.`
其余的书在图书馆。

`La mitad de los asistentes llegó tarde. / La mitad de los asistentes llegaron tarde.`
一半的参与者迟到了。

## 11. El número en las expresiones numéricas / 数字表达中的数

基数词在数上不变，除`uno`有阴性形式`una`和复数形式`unos`/`unas`外。

例如：
`un libro / unos libros`
一本书 / 一些书

`una casa / unas casas`
一所房子 / 一些房子

`veinte libros`
二十本书

`cien casas`
一百所房子

序数词有数和性的变化。

例如：
`primer libro / primeros libros`
第一本书 / 最初几本书

`segunda casa / segundas casas`
第二所房子 / 第二组房子

`tercer capítulo / terceros capítulos`
第三章 / 第三章（复数形式较少用）

分数词和倍数词也有数的变化。

例如：
`medio litro / medios litros`
半升 / 半升（复数形式指多个半升）

`doble cantidad / dobles cantidades`
双倍数量 / 多个双倍数量

`triple salto / triples saltos`
三级跳 / 多个三级跳

## 12. El número en los adverbios y preposiciones / 副词和介词中的数

副词和介词通常在数上不变，尽管有些可能在特殊用法中呈现复数形式。

例如：
`detrás / detrases` (poco usado)
在后面 / 在后面（复数形式很少用）

`cerca / cercas` (poco usado)
附近 / 附近（复数形式很少用）

`ante / antes`
在...之前 / 在...之前（`antes` 是副词，不是 `ante` 的复数）

一些副词短语可能有数的变化。

例如：
`a veces / a vez` (menos frecuente)
有时 / 一次

`en ocasiones / en ocasión`
有时 / 在某个场合

`por momentos / por momento`
时刻 / 在某个时刻

## 13. El número en las interjecciones / 感叹词中的数

感叹词通常在数上不变，尽管有些可能在特殊用法中呈现复数形式。

例如：
`¡Ay! / ¡Ayes!` (poco usado)
哎呀！ / 哎呀！（复数形式很少用）

`¡Oh! / ¡Ohs!` (poco usado)
哦！ / 哦！（复数形式很少用）

`¡Caramba! / ¡Carambas!` (poco usado)
哎呀！ / 哎呀！（复数形式很少用）

## 14. El número en los nombres de colores / 颜色名词中的数

作为可变形容词的颜色名称与名词在数上保持一致。

例如：
`vestido rojo / vestidos rojos`
红裙子 / 多条红裙子

`coche azul / coches azules`
蓝车 / 多辆蓝车

`flor blanca / flores blancas`
白花 / 多朵白花

作为不变名词的颜色名称在复数中不变化。

例如：
`vestido naranja / vestidos naranja`
橙色裙子 / 多条橙色裙子

`coche rosa / coches rosa`
粉色车 / 多辆粉色车

`flor violeta / flores violeta`
紫色花 / 多朵紫色花

## 15. El número en los nombres de letras / 字母名称中的数

字母名称根据一般规则构成复数。

例如：
`la a / las aes`
字母a / 多个字母a

`la be / las bes`
字母b / 多个字母b

`la ce / las ces`
字母c / 多个字母c

`la hache / las haches`
字母h / 多个字母h

一些字母名称有不规则复数形式。

例如：
`la y / las yes` (también `las íes` para la letra `i`)
字母y / 多个字母y

`la u / las úes`
字母u / 多个字母u

## 16. El número en los nombres de notas musicales / 音符名称中的数

## 17. El número en los nombres de marcas y productos / 品牌和产品名称中的数

(NGLE 3.21a) 商标和产品名称在设计同一产品的多个样本时，可以使用复数形式。

例如：
`un Seat / dos Seats`
一辆西雅特汽车 / 两辆西雅特汽车

`un Coca-Cola / dos Coca-Colas`
一瓶可口可乐 / 两瓶可口可乐

`un iPhone / dos iPhones`
一部iPhone / 两部iPhone

(NGLE 3.21b) 一些品牌名称已词汇化为普通名词，并遵循一般的复数规则。

例如：
`un kleenex / dos kleenex` (también `kleenexes`)
一张纸巾 / 两张纸巾

`un típex / dos típex` (不变)
一瓶修正液 / 两瓶修正液

## 18. El número en los nombres de ciencias y disciplinas / 科学和学科名称中的数

(NGLE 3.22a) 以`-ica`结尾的科学和学科名称通常是阴性单数，但可以在特殊语境中使用复数。

例如：
`la física / las físicas` (diferentes ramas)
物理学 / 多个物理学分支

`la matemática / las matemáticas`
数学 / 数学（复数形式更常用）

`la lingüística / las lingüísticas` (diferentes enfoques)
语言学 / 多个语言学流派

(NGLE 3.22b) 一些学科名称优先使用复数形式。

例如：
`las matemáticas`
数学

`las ciencias`
科学

`las letras`
文学

`las artes`
艺术

## 19. El número en los nombres de juegos y deportes / 游戏和运动名称中的数
