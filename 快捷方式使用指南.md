# 🚀 西语语法助手 - 快捷启动指南

## ✅ 快捷方式已创建完成！

你现在有 **3 种方式** 启动西语语法助手：

---

## 方式一：终端命令（推荐）⚡

**打开终端，输入：**
```bash
spanish
```

**停止服务：**
```bash
spanish-stop
```

---

## 方式二：桌面双击 🖱️

**直接双击桌面上的：**
```
启动西语助手.command
```

会自动打开终端并启动服务。

---

## 方式三：在项目目录双击 📁

**打开 Finder，导航到：**
```
/Users/liujingyi/Desktop/vibe-coding/spanish-grammar-agent/
```

**双击文件：**
```
启动西语助手.command
```

---

## 🎯 启动后

1. **等待 3-5 秒**，看到服务启动信息
2. **打开浏览器**，访问：http://localhost:3001
3. **开始学习** 🇪🇸

---

## 🛑 停止服务

### 方式一：命令停止（推荐）
```bash
spanish-stop
```

### 方式二：手动停止
在运行服务的终端窗口按 **`Ctrl + C`**

### 方式三：强制停止
```bash
pkill -f "uvicorn main:app"
pkill -f "next dev"
```

## 💡 提示

- ⚠️ **每次关机重启后需要重新启动**
- ⚠️ **关闭终端窗口会停止服务**
- ⚠️ **不要同时启动多次**（会端口冲突）

---

## 🔧 如果双击文件没反应

**1. 确保文件有执行权限：**
```bash
chmod +x ~/Desktop/启动西语助手.command
```

**2. 右键点击文件 → 打开方式 → 终端**

**3. 系统偏好设置 → 安全性与隐私 → 允许从以下位置下载的 App → 选择"任何来源"**
