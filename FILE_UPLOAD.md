# 文件上传功能更新说明

## 新增功能

现在支持上传多种文件类型进行分析：

### 支持的文件格式

1. **Word 文档** (.docx)
   - 自动提取文本内容
   - 包含表格内容

2. **Markdown 文件** (.md)
   - 保留 Markdown 结构
   - 支持语法高亮

3. **纯文本** (.txt)
   - UTF-8 编码
   - 自动尝试 GBK 兼容

4. **图片文件** (.png, .jpg, .jpeg, .gif, .webp)
   - Base64 编码
   - 图片语义分析功能开发中

### 使用方法

1. 在输入框上方点击"上传文件"按钮
2. 选择文件后，文件会显示在输入区域
3. 可以添加问题或直接点击发送
4. AI 会分析文件内容并回答问题

### API 端点

- **上传文件**: `POST /api/file/upload`
- **分析文件**: `POST /api/file/analyze`

### 技术实现

- 后端使用 FastAPI 的 UploadFile 处理文件上传
- python-docx 用于 Word 文档解析
- 临时文件自动清理
- 前端使用 lucide-react 图标库

### 注意事项

- 文件大小限制由 FastAPI 默认配置控制
- 图片语义分析需要模型与服务层支持视觉消息格式
- 目前主要支持文本内容分析
